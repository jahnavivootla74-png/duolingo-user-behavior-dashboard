# Duolingo User Behavior Dashboard

## Project Overview
This Power BI dashboard analyzes Duolingo user engagement, streak behavior, lesson completion trends, and learning activity patterns using real user review data.

The project focuses on understanding how users interact with language learning platforms and identifying patterns in engagement and consistency.

---

## Objectives
- Analyze user engagement trends
- Track streak consistency behavior
- Measure lesson completion activity
- Visualize learning patterns
- Create an interactive analytical dashboard

---

## Tools Used
- Power BI
- CSV Dataset
- Data Visualization
- Dashboard Design

---
